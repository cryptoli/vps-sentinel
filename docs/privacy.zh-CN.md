# 隐私说明

`vps-sentinel` 默认本地优先运行。未启用通知或面板时，检测数据保存在本机 SQLite 中，不会主动上传到外部服务。

启用通知时，只发送告警或报告需要的摘要字段。启用面板时，agent 会在上传前移除节点 ID、主机 ID、主机名、路径、命令行、原始日志和通用内部网络字段。面板接收端还会再次脱敏。

公开面板页面可以展示已确认的外部攻击源 IP、国家、ASN 和组织，用于公开黑名单和攻击来源透明度；内部主机身份和原始证据不会在公开接口返回。

所有 token、SMTP 凭据、Webhook、Cloudflare API 凭据和 panel shared secret 都应保存在本地配置、systemd 环境文件或平台 secret 中，不应提交到仓库。
