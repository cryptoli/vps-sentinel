# 威胁模型

本项目面向个人 VPS 和小规模服务器集合，重点是尽早发现可疑行为、提供证据和处置建议。

## 覆盖范围

- SSH 爆破、异常登录、账号和密钥变化。
- Web 探查、常见攻击路径、错误爆发和来源聚合。
- 可疑进程、挖矿/GPU 滥用、扫描器、临时目录执行和异常出站行为。
- 持久化入口、关键文件和服务身份漂移。
- 高置信外部攻击源的主动封禁。

## 非目标

本项目不是杀毒软件、漏洞利用框架、密码爆破工具、第三方主机扫描器、C2/后门或绝对安全保证。

## 主要假设

- agent 需要足够权限读取日志、procfs、systemd 和关键配置。
- 用户会为生产业务配置 allowlist，并在启用主动封禁前理解策略。
- 面板只接收脱敏摘要；敏感凭据由本机或平台 secret 管理。
